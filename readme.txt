・解説

・ページ内遷移
-トップページのヘッダー内に、それぞれサイト紹介欄とイラストレーター紹介欄に飛ぶページ内遷移を作成しました。

・疑似要素
-トップページの各イラストレーター紹介ページ欄に、hoverを使用し画像が浮き出てくるアニメーションを導入しました。
-各紹介ページのイラストレーターの名前の下に、beforeを使用し線が伸びるアニメーションを導入しました。
    また、トップページへ戻るボタンとイラストレーターのX(ツイッター)に飛ぶリンクのボタンを、hoverを使用し押し心地をよくしました。



全体的にイラストを目立たせたいのでアニメーションや複雑なデザインをあえてせず、シンプルかつおしゃれだと個人的には思うデザインにしました。
